The goal of pds is to create JSON sidecar files in the Psych-DS format
to describe your data.

## Installation

You can install the development version of pdsbuilder from with:

``` r
devtools::install_github("debruine/pdsbuilder")
```

## Example

``` r
library(pdsbuilder)
# run shiny app

pds_app()
```
